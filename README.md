<h1>Go Touch Grass</h1>
Explore Nature and Complete Outdoor Badges

<h2>Overview</h2>
<h3>Description</h3>
Go Touch Grass is an interactive website project designed and developed to encourage users to connect with nature through engaging achievements. The website provides information about trails in Orlando, and badges to motivates exploration and environmental appreciation. The website is intended to be used before the user heads out onto a trail. 
<br><br>
This project was created for DIG3525: Digital Media Project Management at University of Central Florida (UCF).
